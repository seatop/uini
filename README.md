# markdown 语法
## 1.标题
示例：
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
语法：
```markdown
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```
## 2.字体
示例：

**这是加粗的文字**

*这是斜体的文字*

***这是斜体加粗的文字***

~~这是加删除线的文字~~

语法：

```markdown
**这是加粗的文字**
*这是斜体的文字*
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```
## 3.引用
示例：

>这是引用的内容
>>这是引用的内容
>>>这是引用的内容

语法：

```markdown
>这是引用的内容
>>这是引用的内容
>>>这是引用的内容
```
## 4.分割线
示例：

---
****

语法：

```
---
****
三个或者三个以上的-或者*
```
